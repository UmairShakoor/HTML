Favicon - About

A favicon (favorite icon) is a tiny icon included along with a website, which is displayed in places like the browser's address bar, page tabs and bookmarks menu. Note, however, that most modern browsers replaced the favicon from the address bar by an image indicating whether or not the website is using HTTPS.

Important Explanation about favicon

We can link an img as a favicon in many formats like gif,png,ico,svg,jpeg etc. .ico > .png > .svg The above order means that .ico is considered a best practice when using as favicon bcz it is widely supported by almost all browsers as a favicon (while writing code use .png bcz some code editors don't support .ico file to be opened)and the remaining two formats are supported by browsers as of order .png > .svg ... Don't Stuck to this order, You can also use png(Portable Network Graphic) for favicon according to your convenient..

Note: Favicon is shown is browser by a default dimension of 16*16px but don't use the 16*16px dimensions while creating a favicon or using it as a favicon bcz favicon is not just used in browsers tab but also used as shortcut icon by google.. . Instead, Consider that your favicon must be a multiple of 48px square as : 48*48 px , 96*96 px , 144*144 px and so on.. When you use svg as favicon so, don't worry about it's dimensions

How to build favicon

If you're a (web) designer then , build favicon yourself by using different tools like Canva , PS etc. But if you're a beginner developer or designer then consider free online favcion generator tools like favicon.ico and realfavicongenerator.net Explore both these tools , you will know much better . Favicons are to be used on different places so, we have to ensure their quality at all these place . If you're a beginner then just explore these two tools as above they will carry out most of your work..

While picking or building a favicon ensure :

  1. Min. Dimensions : 96(width)*96(height) px
  2. Ideal Dimensions : 256*256 px
  3. As, favicon load after the main content of webpage so, it is recommended to use a favicon of minimum size or compress it using free online tools (i.e, https://imagecompressor.com). Size for better loading of fav. : 1kb - 100kb
